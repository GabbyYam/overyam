111. Minimum Depth of Binary Tree

解法一：递归遍历

  1. 判空：空时为0

  2. 判断是否为叶子节点：如果是返回1

  3. 判断是否有一边为null：如果只有一边为null的话，应该对其忽略不计，否则会把此处错误地当做最小深度

  4. 递归：最小深度为左右子树的最小深度中更小的那者再+1
