96. Unique Binary Search Trees

解法一：递归法

  • 思路：分别以1-n的任意整数为根，那么左右子树有哪些值是可以确定的，因此可以递归地来做

  • 要点：{4,5,6,7,8}在此处可以等价于{1,2,3,4,5}，因此可以很方便的进行递归

  • 缺点：没有保存递归结果，增加了许多重复的操作，下一个解法考虑对结果进行缓存
