70. Climbing Stairs

解法一：DP

  • 基本情况：一层楼梯有一种方法，两层楼梯两种方法，因此dp[0]=1,dp[1]=2
  • 转移方程：对于大于等于2的层数，dp[n-1]=dp[n-2]+dp[n-3]，即为一般解
