Lexical Analysis（词法分析）

主要任务：

  • 将一段program分割成不同的成分

  • 并标记对应的token（即词性：逻辑运算符、运算符、变量名、关键字、etc）

token的形式：<class,string>,for example:

  • <Id,”foo”>
  • <op,”=”>
  • <Int,”42”>
  • …

标点符号没有token，而是用自己本身表示，比如

  • “:”
  • “(“
  • “;”

正则语言-regular language：

交集（x）、并集（+）、反复（*）
