Compiler/Mooc-Note/Note-01-概述

2020-02-25

概述

高级语言语言 （编译）=> 汇编语言 （汇编）=> 机器语言

结构：类似于语言翻译

  • 分析部分：分析源语言
  • 优化器
  • 综合部分：组织目标语言

**词法分析：**识别各个单词，判断该词的category

  • 词法单元以token的形式呈现 <Kind,Type>
  • 关键词(key word)
  • 常量
  • 运算符 =-+*
  • 界限符 😦{[])
  • 标识符：变量名、方法名、类名、etc

语句通过分析树来进行分析

语义分析：

  • 收集标识符的属性信息
    • Kind
    • Type
    • 储存位置、位置
    • 值
    • 作用域
    • 参数和返回信息
  • 语义检查

中间代码生成：即代码翻译的中间形式

  • 三地址指令（操作符，操作数，被操作数，赋值数）
