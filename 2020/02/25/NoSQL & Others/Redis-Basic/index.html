Redis Basic Usage

string

  • string是可变数组，实现类似于ArrayList，有冗余空间

  • 小于1M，扩容加倍

  • 大于1M，扩容1M

list

  • 类似LinkedList，删改$O(1)$，索引$O(n)$
  • 弹出最后一个元素后，list被删除
  • 常用作异步队列

hash

  • 类似HashMap，数组+链表
  • value只能为string
  • 渐进式rehash

set

  • 对标HashSet，无序键值对，value为null

zset

  • SortedSet和HashMap的结合，value唯一，自带权重
  • 内部用==跳跃列表==实现

容器类的通用规则

容器类指的是：list,set,hash,zset

  • create if not exist
  • drop if no elements
